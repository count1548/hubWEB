호서대학교에는 학생들에게 제공되는 버스와 관련하여 예약 및 각종 편의 기능을 제공하는 애플리케이션인 HUB와 연동하여<br/>
각종 관리기능을 제공되는 웹 페이지 개발

## 기능
- 관리자 로그인
- Application Push 기능
- 공지 관리 [CRUD]
- 통학버스
	- 노선관리 [CRUD]
	- 위치조회
	- 일정관리[CRUD]
	- 티켓관리 [RD] & Download [Excel]
- 셔틀버스
	- 노선관리 [CRUD]
	- 위치조회
	- 일정관리 [CRUD]
	- Payback (셔틀요금환급) 조회
	- 승하차조회
- 정류장 관리 (셔틀/통학)
- 버스위치로그 조회
- 단말기(NFC) 관리 [CRUD]
- 사용자(학생) 관리 [RU]
